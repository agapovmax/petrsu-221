# Фиксация ошибок

## Шаг 1
```c
main.c:9:9: фатальная ошибка: studio.h: Нет такого файла или каталога
 #include<studio.h>
         ^~~~~~~~~~
компиляция прервана.
```

Такого заголовочного файла не существует. Ищем возможные в `/usr/include/`
```c
#include<studio.h>
```

Верная строка
```c
include <stdio.h>
```

## Шаг 2
```c
main.c: В функции «Main»:
main.c:24:5: ошибка: expected «;» before «printf»
     printf("Введите число квартир на каждом этаже: ");
     ^~~~~~
make: *** [Makefile:6: main.o] Ошибка 1
```

Отсутствует символ ";"

Верная строка
```c
scanf("%d", &flat_number);
```

Похоже что имя функции **main** задано с заглавной буквы


## Шаг 4
После включения дополнительных ключей
```c
main.c: В функции «main»:
main.c:28:12: предупреждение: слишком много аргументов для форматирования [-Wformat-extra-args]
     printf("Вам нужно подняться на %d этаж\n",
            ^~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
gcc -g -O0 -o task2 main.o
```

Вывод должен содержать одну переменную %d, в после запятой перечислено действие с тремя переменными. Надо вынести расчёт заранее
```c
    printf("Вам нужно подняться на %d этаж\n",
           flat_number / flats_per_floor, flat_number);
```

Верная строка

```c
    flat_number = flat_number / flats_per_floor;
    printf("Вам нужно подняться на %d этаж\n",
           flat_number);
```

## Шаг 5

После добавления ключей педанта
```c
main.c:13:5: ошибка: комментарии в стиле C++ не разрешены в ISO C90
     // Номер квартиры
     ^
main.c:13:5: ошибка: (об этом будет сообщено только один раз для каждого файла)
make: *** [Makefile:6: main.o] Ошибка 1
```

C++ поддерживает как комментарии в стиле C (/* комментарий */), так и однострочные (//). Сделаем на "старый" лад

Верная строка
```c
/* Номер квартиры */
```


## Возникшие вопросы

1. Неужели можно указывать заголовочный файл без пробела?!

Можно. Даже если указать не в <> в путь до заголовочного файла "".

2. Ввод значения 0,1 или 0.1 ломает программу

Это int переменные, что поделать. Необходима проверка ввода.

3. Деление на "0"

Добавить проверку ввода *scanf* через *while*
