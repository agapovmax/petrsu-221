* задача 7
  1. Необходимо найти [область определения функции](http://mathprofi.ru/oblast_opredeleniya.html). Это значит, что любые значения x, при которых функция равна нулю - не входят в область определения функции.
  Если это дробь - то знаменатель не должен раняться 0.
  Если корень - подкоренное выражение не отрицательно.
  Если логарифм - больше 0. Если логарифм в знаменателе - то больше нуля и не равен 0.
  Ответ записывается двумя способами:
     D(f)=R\{2,-2} - область определения (D(f)) все действительные числа (R) за исключением множества, состоящего из значений (2, -2)
     Область определения (-∞, -2) U (-2, 2) U (2,+∞)
  2. Четная, нечетная или общего вида.
     Если при подстановкеотрицательоного x функция не меняет свой вид f(-x) = f(x) - это четная (симметричная относительно OY)
     Если меняет f(-x) = -f(x) - нечетная (симметрична относительно начала координат)
     Если f(-x) неравно +-f(x) - общего вида.
  3. Пересечение с осями координат.
  4. Асимптоты графика функции.
     Существует три вида ассимптот - вертикальная, горизонтальная, наклонная.
  
  [Как исследовать функцию и построить её график?](http://www.mathprofi.ru/polnoe_issledovanie_funkcii_i_postroenie_grafika.html)
